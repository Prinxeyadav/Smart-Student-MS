🔒 Data Isolation — How your privacy works

All your student records are stored under your unique user ID in Firestore at path users/{your-uid}/students. Other users cannot access, view, or modify your data even if they use the same app. This is enforced by Firestore Security Rules on the server side.

🎓 University Grade System (10-point)

CGPA 9.0 – 10.0 → O (Outstanding)
CGPA 8.0 – 8.9 → A+ (Excellent)
CGPA 7.0 – 7.9 → A (Very Good)
CGPA 6.0 – 6.9 → B+ (Good)
CGPA 5.0 – 5.9 → B (Above Average)
CGPA 4.0 – 4.9 → C (Average)
CGPA below 4.0 → F (Fail)
